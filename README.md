# kubernetes-context-switch

A simple bash script to allow easy switching of Kubernetes context through prompt.

# Install
**cp k8s-context /usr/local/bin/**
**chmod +x /usr/local/bin/k8s-context**

# Notes
Tested on Mac and Centos 6/7
